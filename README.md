# E-Commerce Application

A modern e-commerce application built with Next.js 14, featuring server-side rendering, GSAP animations, and JWT authentication.

## Features

- **Server-Side Rendering (SSR)** for optimal performance
- **GSAP Animations** on product cards
- **JWT Authentication** with phone number OTP login
- **Responsive Design** matching Figma specifications
- **State Management** with Zustand
- **Clean API Integration** with axios
- **Protected Routes** using middleware

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Animations**: GSAP
- **Authentication**: JWT tokens
- **HTTP Client**: Axios

## Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn
